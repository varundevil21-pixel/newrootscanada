# newrootscanada

A simple static site for NewRoots Canada.

## Project structure

- `index.html`
- `assets/css/style.css`
- `assets/js/app.js`
- `data/cities.json`
- `data/agents.json`

## Usage

Open `index.html` in a browser to view the site. The page loads city and agency data from the JSON files.
